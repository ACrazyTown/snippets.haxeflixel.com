Set the {% api flixel.FlxSprite.alpha %} property of your {% api flixel.FlxSprite %} to change the opacity of your sprite. Values can be from `0` to `1`.

```haxe
// sprite's alpha will be set to 0.5 (50% opacity)
sprite.alpha = 0.5;
```
