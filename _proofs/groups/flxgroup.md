{% api flixel.group.FlxGroup %} is a way to separate, and layer {% api flixel.FlxObject %}s. This can be useful to 'force' your objects' draw-order, or to help with overlap/collision detection, etc. 

```haxe
// add sprite to group
group.add(sprite);
```
